Proyecto Intedrador Porfolio Web Full Stack/#Yo Programo - Argentina Programa 4.0 - Mayo 2023

Objetivo: desarrollar una aplicación web full stack, que muestre, mis datos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías. Esta aplicación deberá ser de arquitectura distribuida y contener un diseño de interfaz de usuario (front end) que muestre la información, una base de datos que almacene los datos antes mencionados y debe contar con las APIs necesarias para proveer a través de internet la información (back end) etc.
